# 🎡 sujoy-sikder-3d-carousel Portfolio
A visually stunning, interactive 3D carousel built with HTML, CSS, and JavaScript. This project features smooth animations, mouse/touch controls, and media support (Images, Video, and Audio).

## ✨ Features
- **3D Perspective:** Real-time 3D depth and reflection effects.
- **Interactive Controls:** Drag to rotate, mouse wheel to zoom (change radius).
- **Responsive Design:** Automatically adjusts size for Mobile, Tablet, and Desktop.
- **Multi-Media Support:** Supports rotating images and videos simultaneously.
- **Background Music:** Auto-play audio support for an immersive experience.

## 🚀 How to Use
1. Clone this repository or download the ZIP file.
2. Ensure your media files (1.jpg, 2.jpg... video.mp4, song.mp3) are in the same folder.
3. Open `index.html` in any modern web browser.

## 🛠 Technologies Used
- **HTML5:** Semantic structure.
- **CSS3:** 3D transforms, keyframe animations, and reflections.
- **JavaScript (ES6):** Dynamic sizing, rotation logic, and pointer event handling.
